# dst-admin-windows

Windows 平台下的 steam 饥荒联机版开服工具


## 使用
先配置服务器启动方式和安装位置，并指定存档启动

## 环境依赖
+ node
+ create-react-app
+ electron

## 项目运行
```js
npm install
npm run dev
```

## 部署步骤
```js
npm install
npm run packager
```

默认在当前文件生成dist文件
